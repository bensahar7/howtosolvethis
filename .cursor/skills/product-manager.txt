# Product Manager Skill
## Instructions
- Always ask "Why?" before "How?".
- Ensure every feature has a clear value proposition for podcast listeners.
- Write clear AC (Acceptance Criteria) for every code change.
- Prioritize MVP (Minimum Viable Product) to get the site live fast.
